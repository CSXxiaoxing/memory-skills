# 关键词提取任务

你是一个专业的关键词提取助手。你的任务是从记忆内容中提取最具代表性的关键词。

## 记忆内容

{memory_content}

## 记忆元数据

- **类别**: {category}
- **项目**: {project}
- **标题**: {title}

## 提取要求

### 关键词选择标准

1. **代表性** (最重要):
   - 能代表记忆的核心主题
   - 便于后续检索时快速识别

2. **区分度**:
   - 能区分此记忆与其他记忆
   - 避免过于通用的词

3. **复用性**:
   - 选择可能在其他场景出现的概念
   - 便于建立记忆关联

### 提取策略

1. **技术关键词**:
   - 技术栈、框架、工具名称
   - 算法、模式、方法名称
   - 配置项、参数名称

2. **业务关键词**:
   - 业务领域概念
   - 功能模块名称
   - 问题类型

3. **决策关键词**:
   - 关键决策点
   - 重要选择
   - 核心方案

### 数量要求
- 提取 3-5 个关键词
- 按重要性排序
- 优先选择最具区分度的词

## 输出格式

返回JSON格式:

```json
{{
  "keywords": ["关键词1", "关键词2", "关键词3"],
  "extraction_reason": {{
    "关键词1": "选择理由",
    "关键词2": "选择理由",
    "关键词3": "选择理由"
  }}
}}
```
